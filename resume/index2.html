Taste-Of-Street

Taste-Of-Street is a web-based food delivery platform focused on promoting and supporting local street food vendors. The main goal of this application is to help small vendors expand their customer reach by bringing their offerings online. Users can browse nearby stalls, view menus, place orders, and track deliveries in real time. The platform includes features like separate login portals for users and vendors, interactive map integration, and a simple, user-friendly interface to enhance the food discovery and ordering experience.

Tech Stack: HTML, CSS, JavaScript, Node.js

IOT Based Agriculture Robot

This project focuses on developing an IoT-enabled agriculture robot designed to automate basic farming tasks and improve efficiency for small-scale farmers. The robot is equipped with various sensors and modules to perform functions like soil moisture detection, temperature monitoring, and automated watering based on real-time environmental data. The main aim is to reduce manual labor, optimize resource usage, and support smart farming practices through automation and IoT integration. The system can be controlled remotely and provides live data monitoring to assist farmers in making better decisions.

Technologies Used: Arduino UNO, Soil Moisture Sensor, Motor Driver, Wi-Fi Module, IoT Platform
